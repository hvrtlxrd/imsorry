# Любовное послание

Сайт с любовным посланием, созданный с использованием Bootstrap 5.

## Особенности
- Адаптивный дизайн для всех устройств
- Красивый фон с изображением котика
- Анимированный текст
- Полностью адаптирован под мобильные устройства

## Технологии
- HTML5
- Bootstrap 5.3.0
- CSS
- Адаптивный дизайн

## Как запустить локально
1. Клонируйте репозиторий
2. Откройте файл index.html в браузере
